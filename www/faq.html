    Why the Brenda account is needed?

    The query is made by Brenda SOAP which requires a Brenda account. Because of the possibility of more than one query at the same time we use the personal account for that.

    What can we do with your Brenda account?

    The Brenda is used just to make every query. We also defines your user to separete your generated tables from other tables. But, we can access your password for anything else. If you doubt it, you can see the Java code that make every query: Brenda SOAP Github repository To make it clear, if we could access to your password and Brenda account for personal use, we would only be able to request a change of password (but cannot changing without your mail) and make queries in Brenda.
