    • Acknowlegments

    This application was development as a project of CeBiB (Centre for Biotechnology and Bioengineering). And it couldn't be done without the following packages and tools.

    For the interactive web site

    The whole implementation of the web site was done using the R lenguage project ( R Core Team, 2018 ) using the software for computing environment R Studio .

    Fundamental for the graphical interface and website logic are the R packages Shiny ( Chang et. al, 2018 ) and Shiny Dashboard ( Chang & Riberio, 2018 ).

    To increase the input types, do notification errors and generate help tooltip, they were used, respectively, the following R packages: Shiny Widgets ( Perrier et al., 2018 ), Shiny Alert ( Attali & Edwards, 2018 ) and ShinyBS ( Bailey, 2015 ).

    To complement the website functionality, such as hide/show elements, disable/enable and activate from server was used the R package Shinyjs ( Attali, 2018 ).

    For the SOAP queries logic

    To do query through SOAP, it was develepment a Java Project . This Java program was written using IntelliJ an IDE for JVM develped by Jet Brains .

    SOAP web service was successfully implemented thanks to the Apache Axis library ( Irani and Basha, 2001 ).

    To communicate SOAP Java with R based web site was used the package rJava ( Urbanek, 2018 ).

    For the Data Tables

    Data table has a better visualization and proccessing using DT ( Xie et al., 2018 ) and data.table ( Dowle and Srinivasan, 2019 ).

    Also, to work on the tables was used rlist ( Ren, 2016 ) for process list and plytr ( Wickham, 2011 ) for advanced table function.

    For the Visualization

    The graphs are generated using the packages ggplot2 ( Wickham, 2016 ) and plotly ( Sievert, 2018 ).

    For Processing of the Text

    Biological text, such as aa sequence on FASTA format, is used biostrings ( Pagès, 2019 ) and for more string function stringr ( Wickham, 2019 ) is used. To be access to the clipboard is used rclipboard ( Bihorel, 2017 ).

    For calculations

    Clustering analyse was implemented using the base R function and the DBSCAN package ( Hahsler and Piekenbrock, 2018 ).

    For generate Phylogeny

    This function is not implemented yet. But, at the time, packages tath were using are taxize ( Chamberlain et al., 2019 ) and ggtree ( Yu et al., 2017 ).
